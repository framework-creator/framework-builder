# Example Frameworks

This folder contains example frameworks generated by the Framework Builder methodology.

Each example demonstrates the five-layer DNA architecture applied to a real domain.

## Contributing

Community contributions welcome. If you've built a framework using this methodology, submit a PR with your framework as a new `.md` file in this folder.

Include a brief header noting the domain and what problem the framework solves.

More examples and pre-built frameworks available at [whatisaframework.com](https://whatisaframework.com).
