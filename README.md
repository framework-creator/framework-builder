# Framework Builder

A systematic methodology for building reusable, agent-readable frameworks from any domain expertise.

Built by [Mike Goetz](https://ragedesigner.com). 600+ frameworks across industries since 2024.

## What This Is

A single `SKILL.md` file that teaches AI agents (and humans) how to build frameworks. It works in Claude Code, Claude Projects, Cursor, ChatGPT, Copilot, or any environment that supports skills or custom instructions.

The methodology produces five-layer frameworks that both agents and humans can follow for consistent results across any domain.

## Why It Exists

The AI ecosystem is converging on "skills" (markdown files that give AI context for specific tasks). Individual skills are useful, but they don't compound without systematic methodology.

This skill teaches the meta-methodology: how to generate frameworks across any domain. While everyone else is trading individual skills like baseball cards, this teaches you how to build the card factory.

Built from experience creating 600+ frameworks across federal contracting, sales methodology, medical advocacy, video production, content automation, and dozens of other industries.

## How to Use It

1. Drop `SKILL.md` into your Claude Project, Cursor rules, or any AI skill directory
2. Tell your AI: "Build a framework for [your domain/problem]"
3. The AI uses the five-layer DNA architecture to produce a complete, reusable framework

That's it. One file, any domain, consistent results.

## The Five Layers

Every framework generated by this methodology follows the same DNA:

- **Layer 1: Principles Foundation** - The WHY. Evidence-backed core truths, not opinions.
- **Layer 2: Systematic Approach** - The HOW. Reasoning steps with decision logic, not just procedures.
- **Layer 3: Force Multipliers** - The LEVERAGE. What creates outsized results from small inputs.
- **Layer 4: Success Metrics** - The MEASUREMENT. What good looks like, and what broken looks like.
- **Layer 5: Implementation Guidance** - The DEPLOYMENT. Edge cases, context adaptation, common mistakes.
