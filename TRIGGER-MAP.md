# Trigger Map — When to Reach for Each Framework

The fastest way to get value from this library is knowing which framework to grab before you open it. Use this map.

---

## Socratic Stress Test
Use this when:
- You're about to make a big decision and want to know if you're bullshitting yourself
- Someone's pushing a strong opinion and you want to test how solid it actually is
- You're stuck in a debate and need to surface the real disagreement
- You feel unusually confident about something and want to stress-test it
- You're evaluating a belief that's emotionally loaded

---

## Prompt Archaeology
Use this when:
- The output you're getting from an AI is consistently off or weird
- You have an old prompt that used to work but doesn't anymore
- You're inheriting someone else's messy prompt and need to clean it up
- You want to turn a "pretty good" prompt into a genuinely great one
- You're debugging why an AI keeps misunderstanding your intent

---

## First Principles Compression
Use this when:
- You're trying to learn something genuinely hard and it's not clicking
- You need to explain a complex topic clearly to someone else
- You're solving a problem that feels overly complicated
- You want to strip a strategy, product, or idea down to its cleanest form
- You're preparing for a presentation or deep work session on a messy topic

---

## Decision Lattice
Use this when:
- You have a decision with multiple variables pulling in different directions
- The stakes are high and going with your gut feels irresponsible
- You're choosing between career moves, investments, or big life changes
- You need to compare options that each have different pros, cons, and risks
- You're helping someone else think through a complex choice

---

## Feedback Loop Optimizer
Use this when:
- You're trying to get better at something but progress feels painfully slow
- You're building a new habit and want to make sure you're actually improving
- You want to level up a skill faster than normal
- A project or process keeps producing the same mistakes over and over
- You're managing your own growth and want measurable weekly improvement

---

## Contradiction Resolver
Use this when:
- Two frameworks give you conflicting advice and you don't know which one wins
- You're building your personal framework hierarchy and need to establish priorities
- You're stuck in decision paralysis because two valid approaches disagree
- You want to debug stalled progress where competing principles are at war

---

**Meta-trigger:** If two frameworks ever pull you in opposite directions, run the Contradiction Resolver before doing anything else.
